Ogreのシステム管理ロジックドライバ置場です。 以下のルールに従って、ドライバプラグインを作成してください。

  • ファイル名は "[00-99]$id.csp"
  • クラス名は "plugin_grn_common_system_[00-99]$id"
  • "grn/system_logic.csp" で定義されているクラス GRN_SystemLogicBase を継承、実装する
  • コンストラクタは以下のように作成する 
    
    function plugin_grn_common_system_[00-99]$id()

    {

        parent::GRN_SystemLogicBase( '$id' );

    }
