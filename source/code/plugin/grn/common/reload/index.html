ここは reloadフック関数の置場です。reloadフック関数は以下の規則を守って作成してください。

  • ファイル名にはアプリケーション名を使用してください。(また、ファイル名規約に従い全て小文字にしてください。）
  • 複数作成する場合はアプリケーション名_XXXXとします。アプリケーションが利用可能でないとコールされません。
  • システムのリロードはファイル名をcommon_xxxとしてください。
  • フック関数名は "plugin_grn_common_reload_(ファイル名から拡張子を除いたもの)" です。
  • 引数はアプリケーションIDをとります。
  • フック関数の返り値は使用されません。
例: サンプル 

function plugin_grn_reload_hoge( $id )
{
    // $idは使用しない場合スキップしてください。
    fuga();
}
