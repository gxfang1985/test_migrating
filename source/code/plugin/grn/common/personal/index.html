Ogreの個人設定ロジックドライバ置場です。 以下のルールに従って、ドライバプラグインを作成してください。

  • ファイル名は "[00-99]$id.csp"
  • クラス名は "plugin_grn_common_personal_[00-99]$id"
  • "grn/personal_logic.csp" で定義されているクラス GRN_PersonalLogicBase を継承、実装する
  • コンストラクタは以下のように作成する 
    
    function plugin_grn_common_personal_[00-99]$id()

    {

        parent::GRN_PersonalLogicBase( '$id' );

    }
