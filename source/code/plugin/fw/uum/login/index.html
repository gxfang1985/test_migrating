ここは UUM のログイントリガー関数の置場です。ログイントリガーは以下の規則を守って作成してください。

  • ファイル名は自由です。ただし、ファイル名規約に従い全て小文字にしてください。
  • トリガー関数名は "plugin_fw_uum_login_(ファイル名から拡張子を除いたもの)" です。
  • トリガー関数の引数は、「ログインに成功したユーザー」の CB_User クラスのインスタンスとなります。
  • トリガー関数の返り値は使用されません。
例: ユーザーの個人設定をロケールに反映する 
require_once( 'fw/i18n.csp' );
function plugin_fw_uum_login_hoge( $user )
{
    $i18n = CB_I18N::getInstance();
    $i18n->setCurrentLocale( get_locale_for_user( $user ) );
}
