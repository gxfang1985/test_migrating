ここは UUM のログアウトトリガー関数の置場です。ログアウトトリガーは以下の規則を守って作成してください。

  • ファイル名は自由です。ただし、ファイル名規約に従い全て小文字にしてください。
  • トリガー関数名は "plugin_fw_uum_logout_(ファイル名から拡張子を除いたもの)" です。
  • トリガー関数の引数は、「ログアウト済みユーザー」の CB_User クラスのインスタンスとなります。そのため、内部でgetLoginUserで取り直してもFALSEになります。
  • トリガー関数の返り値は使用されません。 
function plugin_fw_uum_logout_hoge( $user )
{
    ...
}
