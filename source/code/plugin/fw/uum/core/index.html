ここは UUM Core用の各種トリガー置場です。 トリガーは以下の命名規則で作成して設置してください。

ユーザー追加トリガー
"add_user_*.csp" というファイル名で作成。トリガー関数名は "plugin_fw_uum_core_add_user_*" です。
ユーザー削除トリガー
"remove_user_*.csp" というファイル名で作成。トリガー関数名は "plugin_fw_uum_core_remove_user_*" です。
グループ追加トリガー
"add_group_*.csp" というファイル名で作成。トリガー関数名は "plugin_fw_uum_core_add_group_*" です。
グループ削除トリガー
"remove_group_*.csp" というファイル名で作成。トリガー関数名は "plugin_fw_uum_core_remove_group_*" です。
ロール追加トリガー
"add_role_*.csp" というファイル名で作成。トリガー関数名は "plugin_fw_uum_core_add_role_*" です。
ロール削除トリガー
"remove_role_*.csp" というファイル名で作成。トリガー関数名は "plugin_fw_uum_core_remove_role_*" です。
