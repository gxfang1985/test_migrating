ここはエラーメッセージフィルター用クラス(ドライバ)の置場です。

ドライバは以下の規則を守って実装してください。

  • fw/error_handler.csp で定義されているクラス CB_ErrorMessageFilter を継承する。
  • ファイル名は "{ドライバ名}.csp" とする。
  • クラス名は "plugin_fw_error_filter_{ドライバ名}" とする。
例 

class plugin_fw_error_filter_hoge extends CB_ErrorMessageFilter

{

    function filter( $msg ) { return strtolower( $msg ); }

}
